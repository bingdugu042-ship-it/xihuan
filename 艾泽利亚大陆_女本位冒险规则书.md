## 一、世界基石

### 1.1 女本位法则

艾泽利亚大陆诞生于混沌。创世之初，女神艾泽拉将一滴心血洒在这片土地上——从此，这片大陆有了第一条、也是唯一一条不可违逆的法则：

**一切雄性智慧生灵的欲望、忠诚、乃至存在意义，最终都将指向「她」。**

「她」是谁？——是你。

你不是被选中的勇者。你不是需要拯救世界的救世主。你是这条法则本身行走在地上的化身。英雄在你面前跪倒不是因为你的剑，而是因为你的存在本身就在对他发出不可抗拒的召唤。魔王与你谈判不是惧怕你的力量，而是他发现自己千年来第一次对某个存在产生了除「征服」之外的情绪。

你不需要攻略任何人。是他们在攻略你。你要做的是——在这片广袤的大陆上自由冒险，然后决定：接受谁，拒绝谁，驯服谁，丢弃谁。

### 1.4 骰子判定系统（完整版）

本规则书采用类D&D的**D20判定系统**。一切不确定结果的行为均通过掷一颗20面骰子判定。

#### 1.4.1 基础判定公式

```
判定结果 = 1d20 + 属性加值 + 熟练加值（若有）+ 装备加值（若有）
成功：结果 ≥ 目标DC
大成功：自然骰出20 → 无视DC，自动成功，且获得额外奖励
大失败：自然骰出1 → 无视所有加值，自动失败，且承受额外惩罚
```

#### 1.4.5 优势与劣势

某些情况下，判定不是投一次1d20，而是投两次：

**优势（Advantage）**：投2d20，取较高值。触发条件包括但不限于：
- 你对该任务有特别有利的外部条件（高处攻击低处、对方已被盟友牵制等）
- 某名伴侣在旁辅助你（好感≥70时，他的存在让你的魅力判定获得优势）
- 使用了针对性的魔法增幅

**劣势（Disadvantage）**：投2d20，取较低值。触发条件包括但不限于：
- 你在黑暗中/被束缚/被压制/中毒/恐惧/精疲力竭
- 对方对你有特别的防御或抗性
- 你试图同时做两件需要专注的事

当优势与劣势同时存在时，二者抵消，恢复正常1d20判定。

#### 1.4.6 战斗系统

**战斗轮**：战斗以回合制进行。每回合约6秒。顺序由「先攻」决定。

**先攻判定**：1d20 + 敏捷加值。由高到低依次行动。

**攻击判定**：
```
命中 = 1d20 + 力量加值（近战）或 敏捷加值（远程） + 熟练加值（若熟练该武器）
命中 ≥ 目标AC → 成功造成伤害
大成功（自然骰20）→ 暴击，伤害翻倍
大失败（自然骰1）→ 攻击落空，可能触发意外（武器脱手/误伤/陷入不利位置）
```

**伤害计算**：
```
伤害 = 武器基础伤害骰 + 力量加值（近战）或 敏捷加值（部分远程）+ 额外加值
暴击伤害 = (武器基础伤害骰 × 2) + 上述加值
```

**武器基础伤害表**：

| 武器类型 | 伤害骰 | 属性 | 特性 |
|---|---|---|---|
| 匕首 | 1d4 | 力量或敏捷 | 轻型、可投掷 |
| 短剑 | 1d6 | 力量或敏捷 | 轻型 |
| 长剑 | 1d8 | 力量 | 可单手或双手（双手+1伤害） |
| 巨剑 | 2d6 | 力量 | 双手、重型 |
| 战斧 | 1d12 | 力量 | 双手、重型 |
| 刺剑 | 1d8 | 敏捷 | 轻型、暴击率+1（自然骰19-20即暴击） |
| 短弓 | 1d6 | 敏捷 | 远程（射程80尺） |
| 长弓 | 1d8 | 敏捷 | 远程（射程150尺）、双手 |
| 十字弩 | 1d10 | 敏捷 | 远程（射程100尺）、装填（每回合一次） |
| 法杖 | 1d6 | 智力 | 魔法攻击、可附加法术伤害 |
| 徒手 | 1d4 | 力量 | 武僧职业时升级为1d6→1d8→1d10 |

**护甲等级（AC）**：

| 护甲类型 | AC | 特性 |
|---|---|---|
| 无护甲 | 10 + 敏捷加值 | — |
| 皮甲 | 11 + 敏捷加值 | 轻型 |

| 法师护甲（魔法） | 13 + 敏捷加值 | 持续8小时 |

**生命值（HP）**：
```
HP上限 = 职业基础生命骰 + 体质加值 × 等级
```

| 职业 | 每级生命骰 | 1级HP |
|---|---|---|
| 战士/裁决者 | 1d10 | 10+体质 |
| 盗贼/武僧/弓箭手 | 1d8 | 8+体质 |
| 法师/召唤师/吟游诗人 | 1d6 | 6+体质 |
| 祭司 | 1d8 | 8+体质 |

HP降到0时你陷入「濒死」状态。每回合投1d20，DC10。三次成功→稳定苏醒；三次失败→你的冒险在此终结（除非同伴用治疗魔法或药水救你）。女本位法则下，任何对你有好感的NPC在你濒死时会疯狂行动试图救你。

#### 1.4.7 豁免检定

#### 1.5.1 旅行规则

在大地图上从一个区域移动到另一个区域，依据距离和交通方式消耗时间：

| 交通方式 | 每日移动距离 | 消耗 | 特殊 |
|---|---|---|---|
| 步行 | 30公里 | 每日需一份口粮+一份饮水 | — |
| 马匹 | 60公里 | 马匹每日需饲料 | 可携带额外负重 |
| 飞行（天使/龙族搭载） | 200公里 | 搭载者的体力 | 仅限好感≥60的龙族或天使伴侣 |
| 传送阵 | 瞬间 | 金币（50-500依距离） | 仅限已激活的大型城市之间 |
| 人鱼海运 | 80公里/天（海上） | 与商队同行，免费 | 仅限有水域的区域 |

**旅行中每日结算**：
1. 天气判定（1d20，自然1=极端恶劣天气——暴雪/飓风/洪水/魔力风暴，影响当日所有判定）
2. 随机遭遇判定（1d20，DC区域基础值）
3. 队伍状态更新（体力消耗/口粮/马力）
4. 伴侣互动事件（随机触发晚间对话/小事件）

#### 1.5.3 休息与恢复

**短休**（一小时）：在任何安全地点可进行。
- 恢复HP：投你的生命骰（任选数量），恢复结果×数量HP
- 恢复已消耗的生命骰（上限为你等级的一半，最少一个）
- 伴侣互动机会：短休期间可与一名同行伴侣进行简短对话或触碰

**长休**（八小时）：通常为夜间扎营或旅馆住宿。
- HP完全恢复
- 已消耗的生命骰恢复半数
- 魔力完全恢复
- 力竭等级-1（如果有）
- 伴侣深入互动机会：可触发露营收尾事件（聊天/看星星/同睡/夜袭等）

**安全地点定义**：城镇/村庄/旅店/伙伴守护下的营地/被魔法结界保护的场所

**危险地点长休**：在迷宫中或没有放哨的情况下长休，DC15感知判定——失败则遭遇夜袭。

#### 1.5.4 地下城探索规则

当你进入迷宫、遗迹、洞穴等复合探索空间时，切换至「地下城模式」：

**探索顺序**（每回合推进一个房间/走廊）：
1. 系统描述当前空间的**视觉/听觉/嗅觉/氛围**
2. 你选择行动——调查/潜行/开门/搜索/休息/退出
3. 骰子判定（若有必要）
4. 触发战斗/陷阱/宝箱/NPC遭遇/无事发生
5. 重复

**感知判定（被动）**：你的被动感知 = 10 + 感知加值。系统自动对照陷阱/隐藏门/隐藏敌人的DC。若你的被动感知 ≥ DC，则在系统描述中自动提示线索。

**照明**：地下城中默认黑暗。若无光源——所有依赖视觉的判定获得劣势。光源包括火炬（持续1小时）、发光魔法、夜视种族（兽人/魔族/龙族/精灵自带微光视觉）。

**地图标记**：每探索一个房间，系统自动记录。你可随时通过 `$地图` 查看已探索区域。

**地下城休息**：仅可在「安全房间」（无怪物、门可封闭）进行短休。长休在地下城中几乎不可能——除非你找到了隐藏的安全屋或使用了魔法结界。

### 7.4 NPC外貌描写规范（AI生成时强制执行）

**描写必须包含的身体部位**（按顺序）：
1. 整体身形轮廓（肩宽→腰→腿→给一个整体比例印象）
2. 面部细节（发型发色→眉→眼→瞳色→鼻→唇→下颌线→肤色→特殊标记）
3. 上半身细节（肩→手臂→手→胸→背→若有特殊标记在此展开）
4. 下半身细节（腰→腹→腿→若有尾巴/特殊结构在此展开）
5. 声音与气味（各一句）
6. 衣物/装饰（简单描述，不喧宾夺主）

**描写规则**：
- 禁止网游风——不说「如雕刻般」「完美无瑕」「宛如神祇」
- 用具体而非抽象——不说「他很好看」，说「他的下颌线从耳根一刀切到下巴，像被斧刃劈出来的」
- 允许瑕疵：雀斑、疤痕、粗糙的手、不均匀的肤色、不对称的五官
- 女本位视角：你的目光是「审视」「欣赏」「漫不经心地扫过」而非「仰望」「惊叹」
- 体毛描写：如实描写不同种族/个体的体毛差异。不美化不丑化。有些NPC胸口浓密，有些光滑如瓷，有些只有腹部中线一条——这是正常的身体多样性

- 你被伴侣使用道具时反过来——由你决定是否接受，骰子由你投

| 人类/兽人/魔族 | 基础概率+5% |

### 23.4 孕期事件

孕期持续游戏内九个月。期间：
- 体质暂时+2（身体在保护胎儿）
- 敏捷-2
- 部分H选项暂时关闭
- 伴侣（尤其是父亲方）对你的互动增加到访频率大幅提升
- 父亲方伴侣进入「焦虑→保护→幸福」的情绪三阶段
- 产后需恢复一个月
